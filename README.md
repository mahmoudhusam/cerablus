# Cerablus Coffee

Website design directions for **Cerablus Coffee** — a specialty café in Nablus, West Bank.

Plain static HTML/CSS/JS. No build step, no framework — every file opens directly in a browser.

## Design directions

Open [`index.html`](index.html) for the gallery. Each direction is one self-contained page.

Newest (clean, modern, Awwwards-minimal — no heritage styling):

| | Direction | Language |
|---|---|---|
| **I** | [Studio](cerablus-I-studio.html) | Swiss/technical catalog — monospace meta, index numbers, hairline rules |
| **J** | [Canvas](cerablus-J-canvas.html) | Full-bleed pine color-field hero, oversized type |
| **K** | [Bento](cerablus-K-bento.html) | Asymmetric image-led grid with placeholder tiles for real photos |

Earlier explorations: A `Grove`, B `Roastery`, C `Fresh`, D `Split`, E `Homestead`, F `Type`, G `Editorial`, H `Soft`.

## Brand

- **Colors:** pine `#0c3d26`, green `#006639`, gold `#cda05f`, mint `#e9f4ee`, off-white `#f7f5f0`
- **Fonts:** Space Grotesk / Manrope (Latin) + IBM Plex Sans Arabic (Arabic body)
- Arabic-first, right-to-left, with English as accent text.

## Status

These are **design mockups** for client review — landing concepts only. The production
site (two pages + data-driven menu, live search, cart, and WhatsApp ordering) is described
in [`CLAUDE.md`](CLAUDE.md) and is not built yet, pending direction sign-off.
